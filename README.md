# pixelizer
Do its best to pixelize an image

## Simple Example

This is a very simple example of how the script can be used.

```
python pixelizer --input my_image.png \
                 --output pixelized_image.png \
                 --resolution 64x64
```

## Techniques being studied

* Change constrast
* Find edges
* Quantize
* Posterize
